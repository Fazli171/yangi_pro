Albatta, keling, ushbu CSS pesudoklasslarini qisqa va tushunarli tarzda tushuntiraman:

---

:active
- Nima qiladi: Foydalanuvchi bosayotgan elementni tanlaydi (masalan, havolani bosganda).
- Misol: `a:active { color: red; }` ‚Äî Havolani bosganda qizil rangga aylanadi.

---

### **:any-link**
- **Nima qiladi**: `href` atributi bo ªlgan har qanday `<a>` yoki `<area>` elementini tanlaydi.
- **Misol**: `a:any-link { color: blue; }` ‚Äî Barcha havolalarni ko ªk rangga aylantiradi.

---

### **:autofill**
- **Nima qiladi**: Brauzer tomonidan avtomatik to ªldirilgan inputlarni tanlaydi.
- **Misol**: `input:-webkit-autofill { background: yellow; }` ‚Äî Avtomatik to ªldirilgan inputlarning fonini sariq qiladi.

---

### **:checked**
- **Nima qiladi**: Belgilangan (tanlangan) checkbox yoki radio buttonlarni tanlaydi.
- **Misol**: `input:checked { border: 2px solid green; }` ‚Äî Belgilangan inputlarni chetini yashil qiladi.

---

### **:default**
- **Nima qiladi**: Formadagi standart (dastlabki) elementni tanlaydi.
- **Misol**: `input:default { background: lightgray; }` ‚Äî Standart inputlarning fonini och kulrang qiladi.

---

### **:defined**
- **Nima qiladi**: Brauzer tomonidan aniqlangan (standart yoki maxsus) elementlarni tanlaydi.
- **Misol**: `:defined { display: block; }` ‚Äî Aniqlangan elementlarni ko ªrsatadi.

---

### **:dir()**
- **Nima qiladi**: Belgilangan yo ªnalishdagi matnli elementlarni tanlaydi (masalan, `ltr` yoki `rtl`).
- **Misol**: `:dir(rtl) { text-align: right; }` ‚Äî O ªngdan chapga matnli elementlarni o ªngga tekislaydi.

---

### **:disabled**
- **Nima qiladi**: O ªchirilgan (disabled) form elementlarini tanlaydi.
- **Misol**: `input:disabled { opacity: 0.5; }` ‚Äî O ªchirilgan inputlarni yarim shaffof qiladi.

---

### **:empty**
- **Nima qiladi**: Hech qanday bolasi bo ªlmagan elementlarni tanlaydi.
- **Misol**: `div:empty { display: none; }` ‚Äî Bo ªsh `div` larni yashiradi.

---

### **:enabled**
- **Nima qiladi**: Yoqilgan (enabled) form elementlarini tanlaydi.
- **Misol**: `input:enabled { border: 1px solid green; }` ‚Äî Yoqilgan inputlarning chetini yashil qiladi.

---

### **:first**
- **Nima qiladi**: Chop etilgan hujjatning birinchi sahifasini tanlaydi.
- **Misol**: `@page :first { margin: 0; }` ‚Äî Birinchi sahifaning chetlarini olib tashlaydi.

---

### **:first-child**
- **Nima qiladi**: Ota-elementning birinchi bola elementini tanlaydi.
- **Misol**: `li:first-child { font-weight: bold; }` ‚Äî Ro ªyxatdagi birinchi elementni qalin qiladi.

---

### **:first-of-type**
- **Nima qiladi**: Ota-element ichidagi birinchi mos keladigan turdagi elementni tanlaydi.
- **Misol**: `p:first-of-type { color: red; }` ‚Äî Birinchi `<p>` ni qizil rangga aylantiradi.

---

### **:focus**
- **Nima qiladi**: Fokus olgan elementni tanlaydi.
- **Misol**: `input:focus { border-color: blue; }` ‚Äî Fokus olgan inputning chetini ko ªk qiladi.

---

### **:focus-visible**
- **Nima qiladi**: Klaviatura orqali fokus olgan elementni tanlaydi.
- **Misol**: `button:focus-visible { outline: 2px solid orange; }` ‚Äî Klaviatura orqali fokus olgan tugmani ajratib ko ªrsatadi.

---

### **:focus-within**
- **Nima qiladi**: Fokus olgan yoki uning bolalari fokus olgan elementni tanlaydi.
- **Misol**: `form:focus-within { background: lightyellow; }` ‚Äî Forma ichida fokus olgan input bo ªlsa, formaning fonini och sariq qiladi.

---

### **:fullscreen**
- **Nima qiladi**: To ªliq ekran rejimida bo ªlgan elementni tanlaydi.
- **Misol**: `:fullscreen { background: black; }` ‚Äî To ªliq ekran rejimida fonni qora qiladi.

---

### **:has()**
- **Nima qiladi**: Belgilangan bola elementga ega bo ªlgan elementni tanlaydi.
- **Misol**: `div:has(p) { border: 1px solid red; }` ‚Äî Ichida `<p>` bo ªlgan `div` larning chetini qizil qiladi.

---

### **:hover**
- **Nima qiladi**: Sichqoncha kursori element ustiga kelganda tanlaydi.
- **Misol**: `a:hover { text-decoration: underline; }` ‚Äî Havola ustiga kelganda tagiga chiziq chizadi.

---

### **:in-range**
- **Nima qiladi**: Belgilangan oralikda qiymatga ega bo ªlgan inputlarni tanlaydi.
- **Misol**: `input:in-range { border-color: green; }` ‚Äî Oraliqda bo ªlgan inputlarning chetini yashil qiladi.

---

### **:indeterminate**
- **Nima qiladi**: Noaniq holatdagi checkbox yoki progress elementlarini tanlaydi.
- **Misol**: `input:indeterminate { background: gray; }` ‚Äî Noaniq holatdagi inputlarning fonini kulrang qiladi.

---

### **:invalid**
- **Nima qiladi**: Noto ªg ªri qiymatga ega bo ªlgan form elementlarini tanlaydi.
- **Misol**: `input:invalid { border-color: red; }` ‚Äî Noto ªg ªri inputlarning chetini qizil qiladi.

---

### **:is()**
- **Nima qiladi**: Belgilangan selektorlardan birortasiga mos keladigan elementlarni tanlaydi.
- **Misol**: `:is(h1, h2, h3) { color: blue; }` ‚Äî Barcha sarlavhalarni ko ªk rangga aylantiradi.

---

### **:lang()**
- **Nima qiladi**: Belgilangan til atributiga ega bo ªlgan elementlarni tanlaydi.
- **Misol**: `p:lang(fr) { font-style: italic; }` ‚Äî Fransuz tilidagi matnlarni kursiv qiladi.

---

### **:last-child**
- **Nima qiladi**: Ota-elementning oxirgi bola elementini tanlaydi.
- **Misol**: `li:last-child { color: red; }` ‚Äî Ro ªyxatdagi oxirgi elementni qizil rangga aylantiradi.

---

### **:last-of-type**
- **Nima qiladi**: Ota-element ichidagi oxirgi mos keladigan turdagi elementni tanlaydi.
- **Misol**: `p:last-of-type { margin-bottom: 0; }` ‚Äî Oxirgi `<p>` ning pastki chetini olib tashlaydi.

---

### **:left**
- **Nima qiladi**: Chop etilgan hujjatning chap sahifalarini tanlaydi.
- **Misol**: `@page :left { margin-left: 2cm; }` ‚Äî Chap sahifalarning chap chetini kengaytiradi.

---

### **:link**
- **Nima qiladi**: Hali bosilmagan havolalarni tanlaydi.
- **Misol**: `a:link { color: blue; }` ‚Äî Hali bosilmagan havolalarni ko ªk rangga aylantiradi.

---

### **:modal**
- **Nima qiladi**: Modal (o ªz oynasida ko ªrsatiladigan) elementlarni tanlaydi.
- **Misol**: `:modal { background: white; }` ‚Äî Modal elementlarning fonini oq qiladi.

---

### **:not()**
- **Nima qiladi**: Belgilangan selektorga mos kelmaydigan elementlarni tanlaydi.
- **Misol**: `:not(p) { color: green; }` ‚Äî `<p>` dan boshqa barcha elementlarni yashil rangga aylantiradi.

---

### **:nth-child()**
- **Nima qiladi**: Ota-elementning belgilangan tartibdagi bola elementini tanlaydi.
- **Misol**: `li:nth-child(2) { color: red; }` ‚Äî Ro ªyxatdagi ikkinchi elementni qizil rangga aylantiradi.

---

### **:nth-last-child()**
- **Nima qiladi**: Ota-elementning oxiridan boshlab belgilangan tartibdagi bola elementini tanlaydi.
- **Misol**: `li:nth-last-child(2) { color: blue; }` ‚Äî Ro ªyxatdagi oxirgi ikkinchi elementni ko ªk rangga aylantiradi.

---

### **:nth-last-of-type()**
- **Nima qiladi**: Ota-element ichidagi oxiridan boshlab belgilangan turdagi elementni tanlaydi.
- **Misol**: `p:nth-last-of-type(2) { font-weight: bold; }` ‚Äî Oxirgi ikkinchi `<p>` ni qalin qiladi.

---

### **:nth-of-type()**
- **Nima qiladi**: Ota-element ichidagi belgilangan turdagi elementni tanlaydi.
- **Misol**: `p:nth-of-type(2) { color: green; }` ‚Äî Ikkinchi `<p>` ni yashil rangga aylantiradi.

---

### **:only-child**
- **Nima qiladi**: Ota-elementning yagona bola elementini tanlaydi.
- **Misol**: `p:only-child { color: purple; }` ‚Äî Yagona bola bo ªlgan `<p>` ni binafsha rangga aylantiradi.

---

### **:only-of-type**
- **Nima qiladi**: Ota-element ichidagi yagona mos keladigan turdagi elementni tanlaydi.
- **Misol**: `p:only-of-type { font-size: 20px; }` ‚Äî Yagona `<p>` ni kattaroq shriftga aylantiradi.

---

### **:optional**
- **Nima qiladi**: `required` atributi bo ªlmagan form elementlarini tanlaydi.
- **Misol**: `input:optional { border: 1px solid gray; }` ‚Äî Majburiy bo ªlmagan inputlarning chetini kulrang qiladi.

---

### **:out-of-range**
- **Nima qiladi**: Belgilangan oralikdan tashqarida qiymatga ega bo ªlgan inputlarni tanlaydi.
- **Misol**: `input:out-of-range { border-color: red; }` ‚Äî Oraliqdan tashqaridagi inputlarning chetini qizil qiladi.

---

### **:placeholder-shown**
- **Nima qiladi**: `placeholder` matni ko ªrsatilayotgan inputlarni tanlaydi.
- **Misol**: `input:placeholder-shown { color: gray; }` ‚Äî `placeholder` ko ªrsatilayotgan inputlarning matnini kulrang qiladi.

---

### **:popover-open**
- **Nima qiladi**: Ochiq popover elementlarini tanlaydi.
- **Misol**: `:popover-open { background: yellow; }` ‚Äî Ochiq popoverlarning fonini sariq qiladi.

---

### **:read-only**
- **Nima qiladi**: `readonly` atributi bo ªlgan inputlarni tanlaydi.
- **Misol**: `input:read-only { background: lightgray; }` ‚Äî Faqat o ªqish uchun inputlarning fonini och kulrang qiladi.

---

### **:read-write**
- **Nima qiladi**: Tahrirlanishi mumkin bo ªlgan inputlarni tanlaydi.
- **Misol**: `input:read-write { border: 1px solid blue; }` ‚Äî Tahrirlanishi mumkin bo ªlgan inputlarning chetini ko ªk qiladi.

---

### **:required**
- **Nima qiladi**: `required` atributi bo ªlgan form elementlarini tanlaydi.
- **Misol**: `input:required { border-color: red; }` ‚Äî Majburiy inputlarning chetini qizil qiladi.

---

### **:right**
- **Nima qiladi**: Chop etilgan hujjatning o ªng sahifalarini tanlaydi.
- **Misol**: `@page :right { margin-right: 2cm; }` ‚Äî O ªng sahifalarning o ªng chetini kengaytiradi.

---

### **:root**
- **Nima qiladi**: Hujjatning ildiz (root) elementini tanlaydi (`<html>`).
- **Misol**: `:root { --main-color: blue; }` ‚Äî Ildiz elementida CSS o ªzgaruvchisini belgilaydi.

---

### **:scope**
- **Nima qiladi**: Belgilangan kontekst (scope) ichidagi elementlarni tanlaydi.
- **Misol**: `:scope { border: 1px solid red; }` ‚Äî Kontekst ichidagi elementlarning chetini qizil qiladi.

---

### **:state()**
- **Nima qiladi**: Maxsus holatdagi elementlarni tanlaydi.
- **Misol**: `:state(checked) { background: green; }` ‚Äî Belgilangan elementlarning fonini yashil qiladi.

---

### **:target**
- **Nima qiladi**: URLdagi fragment identifikatoriga mos keladigan elementni tanlaydi.
- **Misol**: `:target { background: yellow; }` ‚Äî Maqsad elementining fonini sariq qiladi.

---

### **:user-invalid**
- **Nima qiladi**: Foydalanuvchi tomonidan noto ªg ªri qiymat kiritilgan inputlarni tanlaydi.
- **Misol**: `input:user-invalid { border-color: red; }` ‚Äî Noto ªg ªri qiymat kiritilgan inputlarning chetini qizil qiladi.

---

### **:user-valid**
- **Nima qiladi**: Foydalanuvchi tomonidan to ªg ªri qiymat kiritilgan inputlarni tanlaydi.
- **Misol**: `input:user-valid { border-color: green; }` ‚Äî To ªg ªri qiymat kiritilgan inputlarning chetini yashil qiladi.

---

### **:valid**

- **Misol**: `input:valid { border-color: green; }` ‚Äî To ªg ªri qiymatli inputlarning chetini yashil qiladi.

---

### **:visited**

- **Misol**: `a:visited { color: purple; }` ‚Äî Bosilgan havolalarni binafsha rangga aylantiradi.

---

### **:where()**
- **Nima qiladi**: Belgilangan selektorlardan birortasiga mos keladigan elementlarni tanlaydi (lekin selektror og ªirligini o ªzgartirmaydi).
