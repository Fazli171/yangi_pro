Albatta, keling, ushbu CSS pesudoklasslarini qisqa va tushunarli tarzda tushuntiraman:

---

:active
- Nima qiladi: Foydalanuvchi bosayotgan elementni tanlaydi (masalan, havolani bosganda).
- Misol: `a:active { color: red; }` — Havolani bosganda qizil rangga aylanadi.

---

### **:any-link**
- **Nima qiladi**: `href` atributi boʻlgan har qanday `<a>` yoki `<area>` elementini tanlaydi.
- **Misol**: `a:any-link { color: blue; }` — Barcha havolalarni koʻk rangga aylantiradi.

---

### **:autofill**
- **Nima qiladi**: Brauzer tomonidan avtomatik toʻldirilgan inputlarni tanlaydi.
- **Misol**: `input:-webkit-autofill { background: yellow; }` — Avtomatik toʻldirilgan inputlarning fonini sariq qiladi.

---

### **:checked**
- **Nima qiladi**: Belgilangan (tanlangan) checkbox yoki radio buttonlarni tanlaydi.
- **Misol**: `input:checked { border: 2px solid green; }` — Belgilangan inputlarni chetini yashil qiladi.

---

### **:default**
- **Nima qiladi**: Formadagi standart (dastlabki) elementni tanlaydi.
- **Misol**: `input:default { background: lightgray; }` — Standart inputlarning fonini och kulrang qiladi.

---

### **:defined**
- **Nima qiladi**: Brauzer tomonidan aniqlangan (standart yoki maxsus) elementlarni tanlaydi.
- **Misol**: `:defined { display: block; }` — Aniqlangan elementlarni koʻrsatadi.

---

### **:dir()**
- **Nima qiladi**: Belgilangan yoʻnalishdagi matnli elementlarni tanlaydi (masalan, `ltr` yoki `rtl`).
- **Misol**: `:dir(rtl) { text-align: right; }` — Oʻngdan chapga matnli elementlarni oʻngga tekislaydi.

---

### **:disabled**
- **Nima qiladi**: Oʻchirilgan (disabled) form elementlarini tanlaydi.
- **Misol**: `input:disabled { opacity: 0.5; }` — Oʻchirilgan inputlarni yarim shaffof qiladi.

---

### **:empty**
- **Nima qiladi**: Hech qanday bolasi boʻlmagan elementlarni tanlaydi.
- **Misol**: `div:empty { display: none; }` — Boʻsh `div` larni yashiradi.

---

### **:enabled**
- **Nima qiladi**: Yoqilgan (enabled) form elementlarini tanlaydi.
- **Misol**: `input:enabled { border: 1px solid green; }` — Yoqilgan inputlarning chetini yashil qiladi.

---

### **:first**
- **Nima qiladi**: Chop etilgan hujjatning birinchi sahifasini tanlaydi.
- **Misol**: `@page :first { margin: 0; }` — Birinchi sahifaning chetlarini olib tashlaydi.

---

### **:first-child**
- **Nima qiladi**: Ota-elementning birinchi bola elementini tanlaydi.
- **Misol**: `li:first-child { font-weight: bold; }` — Roʻyxatdagi birinchi elementni qalin qiladi.

---

### **:first-of-type**
- **Nima qiladi**: Ota-element ichidagi birinchi mos keladigan turdagi elementni tanlaydi.
- **Misol**: `p:first-of-type { color: red; }` — Birinchi `<p>` ni qizil rangga aylantiradi.

---

### **:focus**
- **Nima qiladi**: Fokus olgan elementni tanlaydi.
- **Misol**: `input:focus { border-color: blue; }` — Fokus olgan inputning chetini koʻk qiladi.

---

### **:focus-visible**
- **Nima qiladi**: Klaviatura orqali fokus olgan elementni tanlaydi.
- **Misol**: `button:focus-visible { outline: 2px solid orange; }` — Klaviatura orqali fokus olgan tugmani ajratib koʻrsatadi.

---

### **:focus-within**
- **Nima qiladi**: Fokus olgan yoki uning bolalari fokus olgan elementni tanlaydi.
- **Misol**: `form:focus-within { background: lightyellow; }` — Forma ichida fokus olgan input boʻlsa, formaning fonini och sariq qiladi.

---

### **:fullscreen**
- **Nima qiladi**: Toʻliq ekran rejimida boʻlgan elementni tanlaydi.
- **Misol**: `:fullscreen { background: black; }` — Toʻliq ekran rejimida fonni qora qiladi.

---

### **:has()**
- **Nima qiladi**: Belgilangan bola elementga ega boʻlgan elementni tanlaydi.
- **Misol**: `div:has(p) { border: 1px solid red; }` — Ichida `<p>` boʻlgan `div` larning chetini qizil qiladi.

---

### **:hover**
- **Nima qiladi**: Sichqoncha kursori element ustiga kelganda tanlaydi.
- **Misol**: `a:hover { text-decoration: underline; }` — Havola ustiga kelganda tagiga chiziq chizadi.

---

### **:in-range**
- **Nima qiladi**: Belgilangan oralikda qiymatga ega boʻlgan inputlarni tanlaydi.
- **Misol**: `input:in-range { border-color: green; }` — Oraliqda boʻlgan inputlarning chetini yashil qiladi.

---

### **:indeterminate**
- **Nima qiladi**: Noaniq holatdagi checkbox yoki progress elementlarini tanlaydi.
- **Misol**: `input:indeterminate { background: gray; }` — Noaniq holatdagi inputlarning fonini kulrang qiladi.

---

### **:invalid**
- **Nima qiladi**: Notoʻgʻri qiymatga ega boʻlgan form elementlarini tanlaydi.
- **Misol**: `input:invalid { border-color: red; }` — Notoʻgʻri inputlarning chetini qizil qiladi.

---

### **:is()**
- **Nima qiladi**: Belgilangan selektorlardan birortasiga mos keladigan elementlarni tanlaydi.
- **Misol**: `:is(h1, h2, h3) { color: blue; }` — Barcha sarlavhalarni koʻk rangga aylantiradi.

---

### **:lang()**
- **Nima qiladi**: Belgilangan til atributiga ega boʻlgan elementlarni tanlaydi.
- **Misol**: `p:lang(fr) { font-style: italic; }` — Fransuz tilidagi matnlarni kursiv qiladi.

---

### **:last-child**
- **Nima qiladi**: Ota-elementning oxirgi bola elementini tanlaydi.
- **Misol**: `li:last-child { color: red; }` — Roʻyxatdagi oxirgi elementni qizil rangga aylantiradi.

---

### **:last-of-type**
- **Nima qiladi**: Ota-element ichidagi oxirgi mos keladigan turdagi elementni tanlaydi.
- **Misol**: `p:last-of-type { margin-bottom: 0; }` — Oxirgi `<p>` ning pastki chetini olib tashlaydi.

---

### **:left**
- **Nima qiladi**: Chop etilgan hujjatning chap sahifalarini tanlaydi.
- **Misol**: `@page :left { margin-left: 2cm; }` — Chap sahifalarning chap chetini kengaytiradi.

---

### **:link**
- **Nima qiladi**: Hali bosilmagan havolalarni tanlaydi.
- **Misol**: `a:link { color: blue; }` — Hali bosilmagan havolalarni koʻk rangga aylantiradi.

---

### **:modal**
- **Nima qiladi**: Modal (oʻz oynasida koʻrsatiladigan) elementlarni tanlaydi.
- **Misol**: `:modal { background: white; }` — Modal elementlarning fonini oq qiladi.

---

### **:not()**
- **Nima qiladi**: Belgilangan selektorga mos kelmaydigan elementlarni tanlaydi.
- **Misol**: `:not(p) { color: green; }` — `<p>` dan boshqa barcha elementlarni yashil rangga aylantiradi.

---

### **:nth-child()**
- **Nima qiladi**: Ota-elementning belgilangan tartibdagi bola elementini tanlaydi.
- **Misol**: `li:nth-child(2) { color: red; }` — Roʻyxatdagi ikkinchi elementni qizil rangga aylantiradi.

---

### **:nth-last-child()**
- **Nima qiladi**: Ota-elementning oxiridan boshlab belgilangan tartibdagi bola elementini tanlaydi.
- **Misol**: `li:nth-last-child(2) { color: blue; }` — Roʻyxatdagi oxirgi ikkinchi elementni koʻk rangga aylantiradi.

---

### **:nth-last-of-type()**
- **Nima qiladi**: Ota-element ichidagi oxiridan boshlab belgilangan turdagi elementni tanlaydi.
- **Misol**: `p:nth-last-of-type(2) { font-weight: bold; }` — Oxirgi ikkinchi `<p>` ni qalin qiladi.

---

### **:nth-of-type()**
- **Nima qiladi**: Ota-element ichidagi belgilangan turdagi elementni tanlaydi.
- **Misol**: `p:nth-of-type(2) { color: green; }` — Ikkinchi `<p>` ni yashil rangga aylantiradi.

---

### **:only-child**
- **Nima qiladi**: Ota-elementning yagona bola elementini tanlaydi.
- **Misol**: `p:only-child { color: purple; }` — Yagona bola boʻlgan `<p>` ni binafsha rangga aylantiradi.

---

### **:only-of-type**
- **Nima qiladi**: Ota-element ichidagi yagona mos keladigan turdagi elementni tanlaydi.
- **Misol**: `p:only-of-type { font-size: 20px; }` — Yagona `<p>` ni kattaroq shriftga aylantiradi.

---

### **:optional**
- **Nima qiladi**: `required` atributi boʻlmagan form elementlarini tanlaydi.
- **Misol**: `input:optional { border: 1px solid gray; }` — Majburiy boʻlmagan inputlarning chetini kulrang qiladi.

---

### **:out-of-range**
- **Nima qiladi**: Belgilangan oralikdan tashqarida qiymatga ega boʻlgan inputlarni tanlaydi.
- **Misol**: `input:out-of-range { border-color: red; }` — Oraliqdan tashqaridagi inputlarning chetini qizil qiladi.

---

### **:placeholder-shown**
- **Nima qiladi**: `placeholder` matni koʻrsatilayotgan inputlarni tanlaydi.
- **Misol**: `input:placeholder-shown { color: gray; }` — `placeholder` koʻrsatilayotgan inputlarning matnini kulrang qiladi.

---

### **:popover-open**
- **Nima qiladi**: Ochiq popover elementlarini tanlaydi.
- **Misol**: `:popover-open { background: yellow; }` — Ochiq popoverlarning fonini sariq qiladi.

---

### **:read-only**
- **Nima qiladi**: `readonly` atributi boʻlgan inputlarni tanlaydi.
- **Misol**: `input:read-only { background: lightgray; }` — Faqat oʻqish uchun inputlarning fonini och kulrang qiladi.

---

### **:read-write**
- **Nima qiladi**: Tahrirlanishi mumkin boʻlgan inputlarni tanlaydi.
- **Misol**: `input:read-write { border: 1px solid blue; }` — Tahrirlanishi mumkin boʻlgan inputlarning chetini koʻk qiladi.

---

### **:required**
- **Nima qiladi**: `required` atributi boʻlgan form elementlarini tanlaydi.
- **Misol**: `input:required { border-color: red; }` — Majburiy inputlarning chetini qizil qiladi.

---

### **:right**
- **Nima qiladi**: Chop etilgan hujjatning oʻng sahifalarini tanlaydi.
- **Misol**: `@page :right { margin-right: 2cm; }` — Oʻng sahifalarning oʻng chetini kengaytiradi.

---

### **:root**
- **Nima qiladi**: Hujjatning ildiz (root) elementini tanlaydi (`<html>`).
- **Misol**: `:root { --main-color: blue; }` — Ildiz elementida CSS oʻzgaruvchisini belgilaydi.

---

### **:scope**
- **Nima qiladi**: Belgilangan kontekst (scope) ichidagi elementlarni tanlaydi.
- **Misol**: `:scope { border: 1px solid red; }` — Kontekst ichidagi elementlarning chetini qizil qiladi.

---

### **:state()**
- **Nima qiladi**: Maxsus holatdagi elementlarni tanlaydi.
- **Misol**: `:state(checked) { background: green; }` — Belgilangan elementlarning fonini yashil qiladi.

---

### **:target**
- **Nima qiladi**: URLdagi fragment identifikatoriga mos keladigan elementni tanlaydi.
- **Misol**: `:target { background: yellow; }` — Maqsad elementining fonini sariq qiladi.

---

### **:user-invalid**
- **Nima qiladi**: Foydalanuvchi tomonidan notoʻgʻri qiymat kiritilgan inputlarni tanlaydi.
- **Misol**: `input:user-invalid { border-color: red; }` — Notoʻgʻri qiymat kiritilgan inputlarning chetini qizil qiladi.

---

### **:user-valid**
- **Nima qiladi**: Foydalanuvchi tomonidan toʻgʻri qiymat kiritilgan inputlarni tanlaydi.
- **Misol**: `input:user-valid { border-color: green; }` — Toʻgʻri qiymat kiritilgan inputlarning chetini yashil qiladi.

---

### **:valid**
- **Nima qiladi**: Toʻgʻri qiymatga ega boʻlgan inputlarni tanlaydi.
- **Misol**: `input:valid { border-color: green; }` — Toʻgʻri qiymatli inputlarning chetini yashil qiladi.

---

### **:visited**
- **Nima qiladi**: Foydalanuvchi bosgan havolalarni tanlaydi.
- **Misol**: `a:visited { color: purple; }` — Bosilgan havolalarni binafsha rangga aylantiradi.

---

### **:where()**
- **Nima qiladi**: Belgilangan selektorlardan birortasiga mos keladigan elementlarni tanlaydi (lekin selektror ogʻirligini oʻzgartirmaydi).
- **Misol**: `:where(h1, h2) { color: blue; }` — Barcha sarlavhalarni koʻk rangga aylantiradi.

---

### Xulosa:
Ushbu pesudoklasslar orqali siz elementlarni turli holatlar va shartlar asosida tanlashingiz mumkin. Bu esa interfeysni yanada dinamik va foydalanuvchilar uchun qulay qilish imkoniyatini beradi. 😊
